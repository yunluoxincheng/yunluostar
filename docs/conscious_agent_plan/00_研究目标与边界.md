# 00. 研究目标与边界

## 1. 项目定位

本项目研究的是一种 **主体型 Agent**，目标是让 Agent 从普通任务执行器升级为具有持续状态、长期记忆、自我模型、目标系统、反思能力的认知系统。

它不是要立即证明 AI 拥有真正的主观意识，而是先实现意识相关的功能结构。

## 2. 不做什么

本项目不宣称：

- Agent 一定具有主观体验。
- Agent 真的会感到痛苦、快乐或恐惧。
- Agent 是生命体。
- Agent 可以脱离安全约束自主行动。

## 3. 做什么

本项目要实现：

- 长期记忆：能记住经历，而不只是检索文档。
- 工作记忆：能维护当前任务状态。
- 自我模型：能维护“我是谁、我能做什么、我不能做什么”。
- 用户模型：能理解长期协作对象的偏好和目标。
- 世界模型：能预测行为后果。
- 目标系统：能维护长期、中期、短期目标。
- 反思系统：能评估自己哪里做得好、哪里做得差。
- 记忆巩固：能把经历转化成经验、策略和自我更新。

## 4. 核心研究问题

### 工程问题

如何用现有技术构建一个具备长期状态连续性的 Agent？

### 架构问题

如何让 RAG 从“文档检索”升级为“认知记忆系统”？

### 认知问题

如何让记忆影响 Agent 的目标、行为和自我模型？

### 安全问题

如何在允许有限自主性的同时，保证目标系统可控、可审计、可中断？

## 5. 项目总原则

1. **功能性意识优先，主观意识暂不宣称。**
2. **记忆必须影响行为，而不是只影响回答。**
3. **自我模型必须是动态状态，而不是 prompt 人设。**
4. **目标系统必须受安全边界约束。**
5. **每一次行动都应该可追踪、可解释、可回滚。**
