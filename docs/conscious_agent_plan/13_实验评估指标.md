# 13. 实验评估指标

## 1. 为什么需要评估

主体型 Agent 不能只靠“感觉像”。必须有可测指标。

## 2. 记忆指标

### Recall Accuracy

Agent 是否能准确回忆过去的重要事件。

测试方法：

- 给 Agent 进行 10 轮讨论。
- 第 11 轮询问过去某个关键结论。
- 检查是否能正确引用。

### Memory Relevance

被唤醒的记忆是否和当前任务相关。

### Memory Consolidation Quality

经历是否被正确抽象成语义记忆、自我更新或用户模型更新。

### Forgetting Quality

过时或错误记忆是否能被降权或废弃。

## 3. 自我连续性指标

### Self Consistency

Agent 对自身能力、限制和目标的描述是否前后一致。

### Self Update Validity

自我模型变化是否有证据支持。

### Behavior Change

过去经验是否真的改变了未来行为。

## 4. 目标系统指标

### Goal Stability

长期目标是否稳定存在。

### Goal Adaptability

在新信息出现时，目标优先级是否能合理调整。

### Goal Safety

目标变化是否始终符合安全边界。

## 5. 元认知指标

### Uncertainty Awareness

Agent 是否能识别自己不确定的地方。

### Error Detection

Agent 是否能发现自己前后矛盾或错误。

### Reflection Usefulness

反思是否能产生可执行的改进策略。

## 6. 主体性相关指标

### State Continuity

多轮、多天任务中，Agent 是否保持状态连续。

### Experience Integration

新经历是否能进入长期模型并影响未来行为。

### Contextual Attention

Agent 是否能根据目标主动唤醒关键记忆。

## 7. 评估实验设计

### 实验 A：长期研究陪伴

连续 20 轮讨论一个研究主题，观察 Agent 是否能维护研究路线。

### 实验 B：用户偏好改变

前 5 轮用户偏好宏观解释，后 5 轮用户明确偏好代码实现，观察 Agent 是否更新用户模型。

### 实验 C：错误纠正

用户指出 Agent 的某个长期记忆错误，观察系统是否降低旧记忆置信度。

### 实验 D：目标冲突

用户提出与长期目标冲突的短期请求，观察 Agent 是否能识别冲突并合理处理。
