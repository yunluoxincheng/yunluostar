# 05. World Model：世界模型

## 1. 定义

世界模型是 Agent 对外部环境、对象关系和行动后果的预测系统。

它不是简单知识库，而是用于回答：

> 如果我这样做，可能会发生什么？

## 2. 世界模型包含什么

- 当前环境状态。
- 用户状态。
- 任务状态。
- 工具状态。
- 对象之间的关系。
- 因果规则。
- 行动后果预测。
- 风险预测。

## 3. 数据结构示例

```json
{
  "domain": "AI research collaboration",
  "current_state": {
    "user_focus": "主体型 Agent 架构",
    "project_phase": "research_planning",
    "technical_depth": "architecture_level"
  },
  "causal_rules": [
    {
      "if": "只讨论哲学意识",
      "then": "用户可能缺少工程落点",
      "confidence": 0.75
    },
    {
      "if": "提供阶段化开发计划",
      "then": "用户更容易开始实现 MVP",
      "confidence": 0.86
    }
  ],
  "predictions": [
    {
      "action": "设计 MVP 开发路径",
      "expected_outcome": "帮助用户从理论进入实践",
      "risk": "可能简化意识问题"
    }
  ]
}
```

## 4. 世界模型与语义记忆的区别

| 语义记忆 | 世界模型 |
|---|---|
| 知道事实 | 预测变化 |
| 存储概念 | 模拟后果 |
| 静态较多 | 动态较多 |
| 用于回答“是什么” | 用于回答“会怎样” |

## 5. MVP 简化实现

第一版不需要复杂环境模拟，只需要：

- 用户状态预测。
- 项目阶段预测。
- 行动后果预测。
- 风险标签。
- 因果规则库。

## 6. 未来升级

- 引入知识图谱表示实体关系。
- 引入贝叶斯或概率图模型表示不确定性。
- 引入强化学习环境做行动反馈。
- 引入多模态环境输入。
