# 01. 总体架构设计

> **架构演进说明：** yunluostar 已完成从 CLI-first agent 研究原型到多平台认知型 Agentic Bot 平台的方向转型。核心认知架构不变，Bot Runtime 成为产品核心，CLI 保留为 admin/debug/developer 工具。当前重点是稳定显式 Bot adapter contract、demo surface、cognitive trace、plugin safety 和 scope isolation。

## 0. Bot 平台架构总览

```text
Bot 适配器层
  ├── Bot API (POST /v1/bot/message)
  ├── Generic HTTP Adapter (active MVP work)
  ├── Demo Surface (scripts/demo-bot-platform.ts; WebChat page optional)
  └── (未来) IM 适配器 (QQ, Telegram, Discord, Slack, ...)

Bot Runtime (核心)
  ├── Agent Controller
  ├── LLM Provider Access
  ├── Embedding Provider Access
  ├── Cognitive Pipeline (记忆、自我模型、用户模型、目标、反思)
  ├── Plugin System (hooks, commands)
  ├── Scoped Persistence (user/workspace/session + platform/conversation/sender)
  └── Audit Logging

Admin/Debug Surface
  └── CLI (yunluo 命令 — admin, debug, developer 工具)
```

## 1. 从工具型 Agent 到主体型 Agent

普通 Agent 常见结构：

```text
User Input
  ↓
Prompt + LLM
  ↓
Tool Call / RAG
  ↓
Answer
```

主体型 Agent 目标结构：

```text
Environment / User Input
  ↓
Input Interpreter
  ↓
Working Memory
  ↓
Cognitive RAG / Long-term Memory
  ↓
Self Model + User Model + World Model
  ↓
Goal / Motivation System
  ↓
Planner
  ↓
Action Executor / Response Generator
  ↓
Metacognition Monitor
  ↓
Reflection + Memory Consolidation
  ↓
Self / Goal / Memory Update
```

## 2. 核心模块

| 模块 | 作用 |
|---|---|
| LLM Core | 语言理解、推理、总结、规划、解释 |
| Cognitive RAG | 认知记忆系统，管理经历、知识、自我记忆 |
| Working Memory | 当前任务状态和注意力焦点 |
| Self Model | Agent 的身份、能力、限制、长期倾向 |
| User Model | 用户长期目标、偏好、知识水平、互动历史 |
| World Model | 环境状态、因果关系、行动后果预测 |
| Goal System | 长期、中期、短期目标管理 |
| Planner | 任务分解、工具选择、行动计划 |
| Tool Executor | 执行工具调用、代码、检索、文件操作等 |
| Metacognition | 置信度检查、风险检查、目标一致性检查 |
| Memory Consolidator | 反思、抽象、遗忘、冲突检测、自我更新 |

## 3. 一次完整交互流程

```text
1. 用户输入
2. 输入理解器解析意图
3. 工作记忆更新当前状态
4. 记忆系统主动唤醒相关记忆
5. 认知整合器融合记忆、自我模型、用户模型
6. 目标系统选择当前优先目标
7. 世界模型预测可能后果
8. 规划器生成行动方案
9. LLM 生成回答或工具调用
10. 元认知模块检查置信度、风险和一致性
11. 输出回答或执行行动
12. 记录本次交互为情节记忆
13. 反思模块总结得失
14. 记忆巩固器更新长期记忆、自我模型和目标优先级
```

## 4. 架构核心变化

从：

```text
检索资料 → 生成回答
```

变为：

```text
经历记录 → 记忆巩固 → 自我更新 → 目标调整 → 行为改变
```

这就是主体型 Agent 的基本方向。
